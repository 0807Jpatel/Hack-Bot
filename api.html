A directory of upcoming hackathons in North America. Open to all students.

Hackalist has a pretty straightforward JSON-based API that contains all the information available on the frontpage.

Do feel free to use it for your own projects. As always, a link back to Hackalist is always appreciated.

If you'd like to submit an update, correction, or addition to the list, anyone can submit a pull request to the JSON file over on GitHub.
Updates and suggestions can also be emailed to me at [javascript protected email address].

API 1.0 ->



Documentation for JSON parameters:
title string
Name of hackathon
url string
URL of website
startDate string
First day of hackathon in form Month, Day
endDate string
Last day of hackathon in form Month, Day
year string
Year of hackathon
city string
City that hackathon is occuring in.
If in United States, use form City, State (abbreviated)
If elsewhere, use form City, State/Province (abbreviated), Country
host string
Name of institution, organization, or company hosting the event
length string
Length of hackaton in hours
size string
Number of anticpated attendees
travel string
Travel reimbursement offered? yes, no, or unknown
prize string
Prize money offered? yes, no, or unknown
highSchoolers string
High school attendance offered? yes, no, or unknown
facebookURL string
URL of Facebook profile
twitterURL string
URL of Twitter profile
googlePlusURL string
URL of Google+ profile
notes string
Any extra notes, such as special requirements to attend
