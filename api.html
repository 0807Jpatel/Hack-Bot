A directory of upcoming hackathons in North America. Open to all students.

Hackalist has a pretty straightforward JSON-based API that contains all the information available on the frontpage.

Feel free to use it for your own projects! As always, a link back to Hackalist is always appreciated :)

If you'd like to submit an update, correction, or addition to the list, free to submit pull requests to the JSON file over on GitHub.
